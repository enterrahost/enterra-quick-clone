Easily clone or clone & edit WooCommerce products with one click.

== Description ==

Enterra Quick Clone adds two handy actions to your WooCommerce product list:
- **Clone**: Instantly duplicates a product (sets status to Draft).
- **Clone & Edit**: Duplicates the product and takes you straight to the edit screen.

Perfect for quickly building out similar product listings, speeding up your workflow, or setting up templates.

**Features:**
- Duplicates title, content, excerpt, taxonomies, and all meta fields
- Clears SKU to prevent conflicts
- Lightweight, fast, and unobtrusive
- Localisation-ready

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/enterra-quick-clone` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to the **Products > All Products** screen and hover over any product to see the new **Clone** and **Clone & Edit** links.

== Frequently Asked Questions ==

= Does this copy images and custom fields? =  
Yes. It duplicates the featured image, product gallery, and all post meta, including custom fields created by other plugins.

= Will this work with variable or grouped products? =  
Currently, this version is optimized for simple products. Future versions will include support for variable and grouped products.

= Can I bulk clone products? =  
Not yet — but we’re working on it!
