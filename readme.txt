Easily clone or clone & edit WooCommerce products or WordPress posts with one click.

== Description ==

Enterra Quick Clone adds two handy actions to your WooCommerce product and WordPress post lists:
- **Clone**: Instantly duplicates an item (sets status to Draft).
- **Clone & Edit**: Duplicates the item and takes you straight to the editor screen.

Perfect for quickly building out similar product listings or blog posts, speeding up your workflow, or setting up reusable templates.

**Features:**
- Duplicates title, content, excerpt, taxonomies, and all meta fields
- Duplicates featured image and gallery images
- Clears SKU (for products) to prevent conflicts
- Supports WordPress posts and WooCommerce products
- Lightweight, fast, and unobtrusive
- Localisation-ready

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/enterra-quick-clone` directory, or install it via the WordPress plugin screen.
2. Activate the plugin through the 'Plugins' menu.
3. Navigate to **Products > All Products** or **Posts > All Posts** and hover over an item to see the **Clone** and **Clone & Edit** links.

== Frequently Asked Questions ==

= Does this copy images and custom fields? =  
Yes. It duplicates the featured image, gallery images, and all post meta — including custom fields added by other plugins.

= Will this work with variable or grouped products? =  
Currently, this version is optimized for simple products. Variable and grouped product support is planned for a future release.

= Can I bulk clone products or posts? =  
Not yet — but it’s on the roadmap.
